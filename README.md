# 🏏 IPL Analysis Dashboard (2008–2025)

## 📊 Project Overview

This project presents an interactive Power BI dashboard analyzing IPL data from 2008 to 2025.

## 🔍 Key Insights

* 🟠 Orange Cap Winner: B Sai Sudharsan (759 runs)
* 🟣 Purple Cap Winner: M Prasidh Krishna (25 wickets)
* 🏆 Champion: Royal Challengers Bangalore
* 📈 Total Matches: 74
* 💥 Total 6s: 1296
* ⚡ Total 4s: 2251

## 📌 Features

* Season-wise filtering
* Points table analysis
* Player performance insights
* Team comparison

## 🛠 Tools Used

* Power BI
* Excel / CSV
* Data Cleaning & DAX
