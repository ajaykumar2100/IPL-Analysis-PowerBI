# 🏏 IPL Analysis Dashboard (2008–2025)

## 📊 Project Overview

This project presents an interactive Power BI dashboard analyzing IPL data from 2008 to 2025.

## 🔍 Key Insights

* 🟠 Orange Cap Winner 
* 🟣 Purple Cap Winner 
* 🏆 Champion 
* 📈 Total Matches 
* 💥 Total 6s 
* ⚡ Total 4s 

## 📌 Features

* Season-wise filtering
* Points table analysis
* Player performance insights
* Team comparison

## 🛠 Tools Used

* Power BI
* Excel / CSV
* Data Cleaning & DAX
